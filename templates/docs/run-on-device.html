How to run a React Native app on your device

We're going to show you how to quickly create a React Native app and run it on your iPhone with Siphon. You will need a Mac to follow this tutorial.

Our command-line tool walks you through all the steps required to provision your device and download any missing dependencies. You don't need to open Xcode or even run the React Native packager.

For now we only support iOS for running an app on your device. Android support is coming soon.

Install the command-line client

To get started with Siphon you will need to download our lightweight command-line tool. Paste the following command into a terminal window:

$ curl https://getsiphon.com/install.sh | sh

The installer script will take a few seconds to complete. Once that's done, check that the tool is working:

$ siphon --help
Read a beginners guide to using the terminal
Create a Siphon account

If you don't have one yet, you will need to quickly create a Siphon account. This is where your apps will be stored.

Create a free account

It only takes a few moments. Come back to this page when you're finished.

Create an app

We're going to need an app to run on the device. The command-line tool can make us a simple template app in one step.

Navigate to a suitable directory (anywhere is fine) and type the following command to create a new app:

$ siphon create my-app

You can name the app anything you like. The tool creates a new directory with same name.

Run the app on your device

First, make sure your iPhone is plugged in to your machine with a USB cable. You're going to need an Apple Developer account to run a development app on your device.

Type the following into your terminal:

$ cd my-app
$ siphon play

That's it! The tool is going to walk you through all of the required steps needed to prepare your machine and provision your device.

You can also run the app on your local machine using the iOS simulator by typing in siphon develop. Learn more about the available Siphon commands.
